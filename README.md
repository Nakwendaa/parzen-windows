# parzen-windows

Parzen windows to classify [iris dataset](https://en.wikipedia.org/wiki/Iris_flower_data_set) by species.

## Installation

Please be sure to use Python 2.7.10

## Usage

To run the program:
```python TP1_3.3.py```
```python TP1_3.4.py```
```python TP1_4.py```
